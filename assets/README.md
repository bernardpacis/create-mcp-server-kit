# Social preview

GitHub "Social preview" helps your repo look good when shared (X/Twitter, Discord, Slack, etc.).

## Included

- `social-preview.svg` (1280×640)

## How to use it on GitHub

1. Open `assets/social-preview.svg` in your browser.
2. Export it to **PNG** (1280×640). Easiest option: take a screenshot and crop to 1280×640.
3. In GitHub repo:
   - **Settings** → **General** → **Social preview** → **Upload an image**
